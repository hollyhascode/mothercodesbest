Tire Chain Finder

This was a fun project where we wanted to setup an aesthetically attractive and easy-to-use tire chain finder. Unfortunately, the tire chains ended up being too pricey to make sense to sell at the company I work for, but we didn't know this before I built out the project.

I converted an excel file to JSON then used XML and jQuery UI autocomplete to create the search. In addition to functionality, I did the page layout (not the photographs though).
